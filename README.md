## 安全注意事项

⚠️ **重要安全提示：**

1. **账户安全**：使用私有 API 可能违反 Instagram 的服务条款。使用风险自负。
2. **凭据保护**：切勿硬编码凭据。使用环境变量或安全存储。
3. **仅使用 HTTPS**：始终使用 HTTPS 进行 API 通信。
4. **令牌安全**：安全存储会话令牌，切勿在客户端代码中暴露它们。
5. **IP 轮换**：考虑对分布式操作使用代理以避免基于 IP 的检测。
